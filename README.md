# EduPath

EduPath is a web platform designed to help high school students explore universities and academic programs.

This project was inspired by a student orientation event organized during the last two years called **WEJEHNI**,
where university students created stands to guide baccalaureate students and answer their questions about different fields of study 
and universities.

EduPath aims to digitalize this experience by providing an online platform where students can discover programs,
compare universities, and receive guidance for choosing their future studies.

## Technologies

Frontend:
- Next.js
- React

Backend:
- Node.js
- Python
